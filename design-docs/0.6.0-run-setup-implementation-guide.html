Add a slider in the Controls tab (similar to existing ones). Only read the value on Reset / startNewRun.

2 Add presets (Lenient / Balanced / Punishing / Custom) 
<!-- In Controls tab, near top -->
<div class="text-[10px] text-slate-400 mb-1">Presets</div>
<div class="flex gap-1 text-[10px] mb-3">
  <button onclick="applyPreset('lenient')">Lenient</button>
  <button onclick="applyPreset('balanced')">Balanced</button>
  <button onclick="applyPreset('punishing')">Punishing</button>
</div>
function applyPreset(type) {
  if (type === 'lenient') {
    hungerRate = 0.03; selfConsumeRelief = 1.5; groundEatRelief = 3.0;
    selfConsumeThreshold = 8; selfConsumeAmount = 0.5; foodSpawnInterval = 5;
  } else if (type === 'balanced') {
    hungerRate = 0.05; selfConsumeRelief = 1; groundEatRelief = 1.9;
    selfConsumeThreshold = 10; selfConsumeAmount = 0.7; foodSpawnInterval = 8;
  } else if (type === 'punishing') {
    hungerRate = 0.09; selfConsumeRelief = 0.6; groundEatRelief = 1.0;
    selfConsumeThreshold = 15; selfConsumeAmount = 1.0; foodSpawnInterval = 12;
  }
  // sync all sliders
  // set a flag so we know user is on Custom if they touch sliders later
}

4 Record setup in run history

When pushing to runHistory, also store the config used for that run.

runHistory.push({
  run: runNumber,
  durationSec: durationSec,
  longestSec: longestSec,
  finalPop: finalPop,
  deaths: deaths,
  initialAgents: numAgents,

Update renderRunHistory() to display the new fields in the list/cards.
