STEP 1 — Add the Flex Layout Wrapper (Most Common Pain Point)

This is the part most people get stuck on.

You need to change your current structure from this:

<div class="max-w-5xl mx-auto">
  <div class="flex items-center gap-4 mb-4"> ... header ... </div>
  <canvas ...></canvas>
  <div class="mt-3 ..."> phase text </div>
</div>

Into this (we wrap the canvas + phase text in a flex row):

<div class="max-w-5xl mx-auto">
  <div class="flex items-center gap-4 mb-4"> ... header ... </div>

  <!-- NEW: flex container starts here -->
  <div class="flex gap-6 items-start">

    <!-- Left column: simulation -->
    <div class="flex-1">
      <canvas ...></canvas>
      <div class="mt-3 ..."> phase text </div>
    </div>

    <!-- Right column: sidebar (we add this in Step 2) -->
    <!-- SIDEBAR WILL GO HERE -->

  </div>
  <!-- flex container ends here -->
</div>
Exact edit you need to make right now:
1. Find the line that says <div class="max-w-5xl mx-auto">
2. After the header div (the one with the title and buttons), insert the new flex opening:
<div class="flex gap-6 items-start">

3. Wrap your <canvas> + the phase text div inside a new <div class="flex-1"> ... </div>

4. After that inner div, add a comment: <!-- SIDEBAR GOES HERE -->

5. Close the flex div before the final </div> of the main container.

Do this change first before adding any sidebar HTML. Test that the page still looks normal (just wider now).

STEP 3 & 4 — Add the JavaScript Wiring + Init Call

Add this function near the bottom of your <script> tag, before the start(); call:

function initHungerSliders() {
  // Hunger Rate
  const sRate = document.getElementById('slider-hunger-rate');
  const vRate = document.getElementById('val-hunger-rate');
  sRate.value = hungerRate;
  sRate.oninput = () => { hungerRate = parseFloat(sRate.value); vRate.textContent = hungerRate.toFixed(3); };

  // Repeat the same pattern for the other 4 sliders...
  // (Self-Consume Relief, Ground Eat Relief, Threshold, Amount)
}

// IMPORTANT: Call it at the very end
initHungerSliders();
Critical: The initHungerSliders(); call must be at the very bottom of the script, after all functions are defined and after the sidebar HTML exists in the DOM.
Common Problems & Fixes
Sliders don't appear or are broken:
→ You probably put the sidebar HTML outside the new flex container. It must be a direct child of <div class="flex gap-6...">.
Sliders appear but don't do anything:
→ The initHungerSliders() function is either not defined, or you are calling it too early (before the sidebar HTML exists).
Page looks broken after adding flex:
→ Make sure you wrapped the canvas + phase text in <div class="flex-1">...</div> and closed it properly.
Console error "hungerRate is not defined":
→ You added the sliders before you added the 5 let variables at the top.
