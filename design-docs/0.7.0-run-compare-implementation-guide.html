1 Make trends show the full current run (no truncation)

Currently the trend arrays are sampled every 30 ticks and capped at 400. To guarantee the whole run is visible from tick 0:

  • Keep the existing sampling cadence.
  • Reset the arrays cleanly on startNewRun so they always represent the full history of *this* run.
  • Update the chart drawing logic to map x-axis across the entire current run length (not just the samples collected so far in a way that truncates view).

2 Capture extra per-run metrics (avg hunger, avg food, avg pop, end condition)

Add lightweight running accumulators (reset every run). Sample alongside the existing trend sampling for consistency.

// New run trackers (add near other run* vars)
let runPopSum = 0;
let runHungerSum = 0;
let runFoodSum = 0;
let runSampleCount = 0;

// In update() — after agents = alive and after food/hunger processing, 
// or inside the existing 30-tick trend sample block:
if (agents.length > 0) {
  runPopSum += agents.length;
  let hSum = 0, fSum = 0;
  for (let a of agents) { hSum += a.hunger; fSum += a.food; }
  runHungerSum += hSum / agents.length;
  runFoodSum += fSum / agents.length;
  runSampleCount++;
}

// In startNewRun()
runPopSum = 0;
runHungerSum = 0;
runFoodSum = 0;
runSampleCount = 0;

Then in recordCurrentRun():

const avgPop = runSampleCount > 0 ? Math.round(runPopSum / runSampleCount) : 0;
const avgHunger = runSampleCount > 0 ? Math.round((runHungerSum / runSampleCount) * 100) / 100 : 0;

3 Remake history tab with richer cards (subtle borders + padding)

Keep the tab title. Restructure the list into card-style entries. Show setup prominently, then key metrics including the new ones.

function renderRunHistory() {
  // ... best-longest stays the same ...

  let html = '';
  for (let i = runHistory.length - 1; i >= 0; i--) {
    const r = runHistory[i];
    const presetBadge = r.preset ? 
      `${r.preset}` : '';
    const setup = r.initialAgents ? `${r.initialAgents} agents` : '';

    html += `
      

        

          Run #${r.run}
          ${r.longestSec}s
        

        

          ${r.durationSec}s • ${r.finalPop}p • ${r.deaths || 0}d
        

        

          ${presetBadge}
          ${setup}
        

        

          avg pop ${r.avgPop || '—'} • avg h ${r.avgHunger || '—'} • avg f ${r.avgFood || '—'}
        

        
ended: ${r.endedBy || '—'}

      
`;
  }
  listEl.innerHTML = html;
}

Use subtle borders + padding as shown. Setup info is easy to scan at the top of each card.
